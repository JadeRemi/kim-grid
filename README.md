# Kim-Nguyen Grid

A simple visual project featuring an animated grid on a canvas, built with React and TypeScript.

## Quick Start

```bash
yarn install
yarn dev
```

## Scripts

- `yarn dev` - Start development server
- `yarn build` - Build for production
- `yarn lint` - Run ESLint
- `yarn typecheck` - Check TypeScript types
- `yarn test` - Run tests

## Features

- 1920x1080 canvas with 60 FPS rendering
- Simple grid visualization with light grey borders on dark grey background
- Real-time FPS display
- Error boundary for stability
